## 为什么这样组织

`.cursor/rules/frontend-coding.md` 是 Cursor IDE 的规则文件，每次 AI 生成代码时**自动注入上下文**，是规范真正生效的地方。

将规范维护在两个文件中会导致内容漂移，因此以该文件为单一来源，本文件仅作索引入口。

> **注意**: 当前项目中 `.cursor/rules/frontend-coding.md` 尚未创建，如需前端规范请联系项目负责人。
